Tags importantes


Tag em: deixa o texto em itálico.

Tag mark:é usada como marca-texto, para destacar algum ponto do texto.

Tag ins: (insert)é usada para sublinhar o texto

Tag sup:Eleva um número ou letra. Ex: 2 | A

Tags semânticas: Possuem significado e expõem isso ao navegador, no momento de renderização.

Tag small: Diminui uma palavra/informação

Tag del: "Deleta" algo.

Link: tag a: coloca o link. você pode adicionar o atributo target=_"blank", que faz com que a página que será aberta, se abra em outra aba de pesquisa, e não na mesma.

Img: é uma tag onde você pode colocar uma imagem ou até o endereço de imagem do próprio servidor sem precisar baixá-la. Pode usar os atributos width e hight para definir a altura e largura.

Atrbuto Alt:Explica de forma mais sinestésica o que significa tal imagem. É um ponto necessário para aa acessibilidade do site.

Tag Figure: Alinham a imagem a uma legenda de imagens. Ele une a tag figcaption.
